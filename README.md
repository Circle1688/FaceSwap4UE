# FaceSwap4UE
用于虚幻引擎的换脸服务

## 项目需求
1. 用户上传一张自己的照片
2. 场景截图，将截图出来的模特模型的脸换成用户上传的

## 运行平台
 - 引擎版本：5.1
 - 操作系统：Windows 10+
 - 显卡：未知
